Программа осуществляет эхо передачу файла через дуплексный файл. Изначальный файл file.txt, результат записывается в output.txt.

При использовании команды 
```
make
```
программа срабатывает, осуществляется передача, в конце выводится затраченное время.
При использовании
```
make test
```
создается файл в 4 Гб и программа работает с ним. В конце выводится время работы, md5 суммы начального и итогового файлов и сообщение, совпали они или нет.
Среднее время по таким тестам около 6,5 секунд.
