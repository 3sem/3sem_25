## Компиляция и запуск
```
make build
make run threads=<...> points=<...>
make clean
```

## Вывод

Оптимальная производительность достигается при использовании 12 потоков, что соответствует количеству физических ядер процессора. Дальнейшее увеличение числа потоков приводит к уменьшению производительности из-за накладных расходов на синхронизацию между потоками, переключения контекста и конкуренции за ресурсы процессора.
